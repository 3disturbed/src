💡 Pro Tips

Getting the Most Out of PixelArtPro:

  • Organize with Layers: Separate different elements (background, character, effects) on different layers for easy editing.
  • Animation Workflow: Create your first pose, duplicate the frame, adjust slightly, and repeat for smooth animations.
  • Use Onion Skin: Enable onion skin with both forward and backward frames visible for best results.
  • Save Frequently: Use "Save Project" to regularly save your work as .pixelart files.
  • Zoom In/Out: Use the zoom slider to alternate between seeing details and the full picture.
  • Color Picker: Quick key I to pick any color directly from the canvas.
  • Right-Click to Erase: Right-click while drawing to erase with the current brush size.
  • Grid Reference: Canvas grid appears automatically when zoomed in enough.

📝 Getting Started Guide

Step 1: Create Your First Canvas

Click "Launch Editor" to open the pixel editor. You'll see a 32×32 pixel canvas with a background layer.

Step 2: Draw Something

Select the Pencil tool and click on the canvas to draw. Use the color picker to change colors. Adjust brush size for different stroke widths.

Step 3: Organize with Layers

Click "+ Layer" to add more layers. Switch between layers by clicking on them in the Layers panel. This keeps different parts of your art separate.

Step 4: Create Animation

Click "+ Frame" to add animation frames. Draw each frame, set the duration, and click Play to preview your animation.

Step 5: Save Your Work

Click "Save" to export your project as a .pixelart file that you can open later. Use "Export PNG" to save as an image file.
